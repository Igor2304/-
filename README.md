# GameBoost Lite

Рабочий Android-проект для Android Studio.

## Что умеет
- Показывает список установленных приложений/игр.
- Запускает выбранную игру через игровой профиль.
- Держит экран включённым во время профиля.
- Может включать режим «Не беспокоить», если пользователь дал доступ.
- Открывает системные настройки для отключения экономии батареи.

## Чего не делает
Без root нельзя честно разгонять CPU/GPU, чистить чужую RAM, менять governor, убивать системные процессы или гарантированно повышать FPS. Такие приложения в Play Market часто просто имитируют буст.

## Как собрать APK
1. Открой папку GameBoostLite в Android Studio.
2. Дождись Gradle Sync.
3. Build > Build APK(s).
4. APK будет в app/build/outputs/apk/debug/.

## Важно
targetSdk стоит 34, потому что на Android 15+ для target 35 Google ограничил прямое управление глобальным DND. Для Play Market позже надо будет переходить на AutomaticZenRule.
